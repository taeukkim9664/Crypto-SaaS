### **Blueprint: 3D 인터랙티브 포트폴리오 웹사이트**

#### **1. 프로젝트 개요**

*   **목표:** 사용자의 기술과 프로젝트를 선보이는 독특하고 몰입감 있는 3D 인터랙티브 웹사이트를 제작합니다. 백엔드 시스템을 연동하여 외부 데이터를 처리하고 동적으로 콘텐츠를 표시합니다.
*   **핵심 기술:** Three.js, Node.js, Express, Chart.js, Public Crypto APIs, Modern Web Standards.
*   **사용자 경험:** 사용자가 스크롤하면서 3D 공간을 탐색하고, 동적으로 업데이트되는 프로젝트 및 실시간 데이터 차트와 같은 콘텐츠를 발견하는 경험을 제공합니다.

#### **2. 개발 우선순위 및 단계별 계획**

**Phase 1: 기본 토대 구축 (완료)**

*   [x] `index.html`, `style.css`, `main.js` 기본 파일 구조 생성
*   [x] Three.js 라이브러리 연동 및 기본 3D 씬 설정

**Phase 2: 3D 세계관 확장 (완료)**

*   [x] 우주 배경, 펭귄 오브제 추가 등 시각적 요소 구성

**Phase 3: 콘텐츠 통합 및 인터랙션 구현 (완료)**

*   [x] 정적 HTML 콘텐츠 구조 추가 및 기본 CSS 스타일링
*   [x] 스크롤 기반 3D 카메라 애니메이션 구현

**Phase 4: 실시간 김치 프리미엄 차트 고도화 (진행 중)**

*   [x] **백엔드 API 확장:**
    *   [x] 쿼리 파라미터를 통해 거래소 및 암호화폐 종목(`BTC`, `ETH` 등) 선택을 완벽하게 지원합니다.
    *   [x] 실시간 환율 API를 연동하여 프리미엄 계산의 정확도를 향상시켰습니다.
*   [ ] **프론트엔드 UI/로직 개선 (종목 선택):**
    *   [ ] `index.html`의 종목 선택 UI를 기존 `<select>` 드롭다운에서 사용자가 요청한 이미지와 같이 클릭 가능한 `<button>` 리스트 방식으로 변경합니다.
    *   [ ] `style.css`에 버튼 UI에 대한 스타일(활성/비활성 상태 포함)을 추가합니다.
    *   [ ] `main.js`에서 각 버튼의 클릭 이벤트를 감지하여, 선택된 종목의 차트를 다시 그리도록 로직을 전면 수정합니다.

**Phase 5: 디자인 및 시각적 완성도 향상**

*   [ ] **모던 CSS 스타일링:** 동적으로 로드된 콘텐츠에 맞춰 CSS를 수정하고 전반적인 디자인을 다듬습니다.
*   [ ] **시각 효과(VFX):** 3D 객체에 마우스를 올렸을 때 빛나는 효과(Glow Effect) 등을 추가하여 인터랙션을 강화합니다.
*   [ ] **반응형 디자인:** 다양한 화면 크기에서 모든 요소가 올바르게 표시되도록 최적화합니다.

**Phase 6: 배포**

*   [ ] 프론트엔드 및 백엔드 서버를 Firebase 호스팅 및 Cloud Functions 등을 통해 배포합니다.
